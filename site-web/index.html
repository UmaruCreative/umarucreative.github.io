PLAN A

Gardez tous les codes de votre site

Contrôle total sur votre site web : Vous gardez TOUT le contrôle de votre site web. Si un jour vous décidez de quitter Umaru Créative, vous pouvez emporter avec vous l'intégralité des codes de votre site.
Assistance et maintenance incluses : profitez d'une période de maintenance de 12 mois pour une tranquillité d'esprit totale.
Idéal pour les personnes connaissant les codes : Ce plan convient particulièrement aux personnes qui maîtrisent les codes et souhaitent garder une totale autonomie sur leur site web.

PLAN B

Accès limité à vos codes

Gestion complète de votre site web : Nous créons votre site web et vous en confions la gestion complète. Cependant, vous n'aurez pas un accès direct aux codes sources.
Assistance pour les modifications : Si vous avez besoin de faire des modifications, nous sommes là pour vous aider et prendre en charge les ajustements nécessaires.
Idéal pour ceux moins à l'aise avec les codes : Ce plan convient parfaitement à ceux qui se sentent moins à l'aise avec les codes, mais qui souhaitent avoir un site web professionnel sans se préoccuper de la maintenance technique.
